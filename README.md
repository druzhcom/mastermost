# Description

Mastermost is a program and a set of tools for creating a bridge between heterogeneous blockchains for transferring information between them.

## Сборка

```bash
npm install
npx hardhat compile
npx hardhat test
npx hardhat node &
npx hardhat run tasks/deploy.js --network localhost
```
